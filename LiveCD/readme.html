Getting Started

The plugin server is designed to turn any machine into a customizable server. Plugins that are included with this distribution of Plugin Server are:

  • Static HTML
  • Chatterbox (Chat Program)
  • PHP (PHP 5 included)
  • Windows Services Dashboard

To install the Plugin Server see Installation instructions below

Installation

To install Plugin Server click here and follow the Install Wizard instructions

NOTE: PHP 5 will be installed along with this version. Directories necessary for the proper use of this program will be created upon install.

Test to make sure the Plugin Server was installed properly by launching the new shortcut on your desktop entitled "Plugin Server Console". Once the server starts, click here or open a web browser and point at the URL: http://localhost:8080

What to do Next

 

Now that the Server has been successfully installed, the plugins directory holds the plugins for the individual capabilities of the server. The plugins folder default location is: C:\Program Files\IT 347 Team 4\Plugin Server\plugins

New plugins can be written and dropped into this directory to increase the functaionalty and capabilities of the Plugin Server. Any plugins that are removed from this directory will be "uninstalled" from the server while it is running. It is not necessary to stop the server to load or unload plugins from this directory.

Uninstalling

To uninstall the Plugin Server from your machine, go to Add/Remove Programs from the Control Panel and select Plugin Server and click the "Remove" button. This will uninstall the program from your machine.
